# Local Cloud

Local-first infrastructure playground: reproducible, traceable, testable, safe.

## Stack
- Floci (AWS emulator)
- k3s (Kubernetes)
- OpenTofu (Infrastructure as Code)
- Pulumi Go (Infrastructure as Code)
- Dagger (CI/CD)
- Prometheus + OpenTelemetry (Observability)
- Hermes (Agent orchestration)
